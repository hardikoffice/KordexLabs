# Monorepo Structure

## Overview
The project uses a monorepo structure to house both the Next.js frontend and the FastAPI backend, allowing for streamlined development and shared configurations where applicable.

## Directory Tree
```text
kordexlabs/
│
├── apps/
│   ├── web/                  # Next.js Frontend
│   │   ├── src/
│   │   │   ├── app/          # Next.js App Router (Pages & Layouts)
│   │   │   ├── components/   # Atomic UI Components (Atoms, Molecules, Organisms)
│   │   │   ├── lib/          # Utilities, Zustand/Redux stores
│   │   │   └── styles/       # Tailwind CSS configurations
│   │   ├── public/           # Static assets
│   │   └── package.json
│   │
│   └── api/                  # FastAPI Backend
│       ├── main.py           # Entry point
│       ├── core/             # Config, security, database setups
│       ├── models/           # SQLAlchemy / SQLModel definitions
│       ├── routers/          # REST and WebSocket endpoints
│       ├── services/         # Business logic, data fetching (yfinance, NewsAPI)
│       └── requirements.txt
│
└── package.json              # Root monorepo config
```
