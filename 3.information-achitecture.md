# Information Architecture

## 1. Site Structure
- **Home (/):**
  - Hero Section
  - Ticker Tape (Real-time Stocks)
  - Trending Blogs (Top 3)
  - Tool of the Week Spotlight
- **AI News (/blogs):**
  - Blog Listing (Filter/Sort by tags)
  - Individual Post (/blogs/[id]): Content, TOC, Progress Bar 
- **AI Tools (/tools):**
  - Tools Listing
  - Comparison Matrix (/tools/compare)
- **Markets (/markets):**
  - AI Stocks & Indices List
  - Stock Chart Details (/markets/[ticker])
- **User Dashboard (/dashboard):**
  - Authentic User Preferences
  - Saved Blogs, Pinned Tools, Favorite Stocks
  - Theme Toggle (Light/Dark)

## 2. Component Hierarchy (Atomic Design)
- **Atoms:** Buttons (glowing effects), Typography tags, Stock up/down arrows, Icons (Lucide React).
- **Molecules:** Stock ticker cards, Blog post thumbnails with hover overlays.
- **Organisms:** Bento-box dashboard grid, Tool comparison matrix table, Interactive stock chart module.
- **Templates:** Overall page layouts (Navbar, Sidebar, Footer, Content Containers).
