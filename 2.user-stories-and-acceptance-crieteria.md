# User Stories & Acceptance Criteria

## Epic 1: User Dashboard & Customization
**User Story:** As a user, I want to customize my dashboard layout so that I can see my most important tools, stocks, and blogs first.
**Acceptance Criteria:**
- User can drag and drop widgets in a bento-grid layout.
- The layout configuration is saved to the database (`dashboard_layout_config`).
- User can toggle between dark and light themes, and the preference is saved (`theme_preference`).

## Epic 2: AI News & Blogs
**User Story:** As an enthusiastic reader, I want an immersive reading experience for AI news with clear progress tracking.
**Acceptance Criteria:**
- Display estimated read time and dynamic progress bar on blog pages.
- Sticky table of contents must be visible.
- Serve heavy blog images via CDN for fast loading.

## Epic 3: AI Tools Compare
**User Story:** As a business leader, I want to compare multiple AI tools easily to make adoption decisions.
**Acceptance Criteria:**
- Horizontal-scrolling comparison matrix rendering attributes like pros, cons, pricing tier.
- Matrix supports comparing at least two tools side-by-side.
- Apply color-coded tags for use cases and limitations.

## Epic 4: Real-time Stock Tracking
**User Story:** As an investor, I want to see real-time updates of AI stock prices to make rapid decisions.
**Acceptance Criteria:**
- Live ticker tape of top AI stocks on the homepage.
- Interactive charts update smoothly in real-time via WebSockets.
- Price changes flash red (down) or green (up) dynamically.
- Hovering over chart displays exact price and timestamp.
