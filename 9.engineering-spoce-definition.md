# Engineering Scope Definition

## 1. In-Scope (MVP Phase)
- **Frontend:**
  - Responsive web application using Next.js.
  - Dark/Light mode toggle with smooth transitions.
  - Bento-box dashboard grid with drag-and-drop customization.
  - Interactive AI Tools comparison matrix.
  - Real-time stock charts using WebSockets and Framer Motion.
  - Blog reading interface with progress bar and sticky TOC.
- **Backend:**
  - RESTful APIs using FastAPI for static and slow-changing data.
  - WebSocket endpoints for streaming real-time market data.
  - Integration with Yahoo Finance/Polygon.io for market data.
  - PostgreSQL database schema for Users, Blogs, Tools, and Stocks.
- **Infrastructure:**
  - Vercel deployment for the Next.js frontend.
  - Render/Railway deployment for the FastAPI backend.
  - CDN integration (Cloudinary) for asset delivery.

## 2. Out-of-Scope (Future Phases)
- Mobile application (iOS/Android) development.
- User-generated blog creation or comments.
- Complex machine learning models for stock prediction.
- Direct brokerage integration for real stock trading.
