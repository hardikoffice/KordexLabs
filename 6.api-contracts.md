# API Contracts

## 1. REST Endpoints (FastAPI)

### Get Trending Blogs
**Endpoint:** `GET /api/blogs/trending`
**Description:** Populates the homepage blog cards with the top 3 trending blogs.
**Response:**
```json
[
  {
    "id": "uuid",
    "title": "String",
    "author": "String",
    "read_time_minutes": "Integer",
    "hero_image_url": "String",
    "tags": ["String"]
  }
]
```

### Get Tools Matrix
**Endpoint:** `GET /api/tools/matrix`
**Query Parameters:** `compare` (Comma-separated list of tool IDs/names, e.g., `ToolA,ToolB`)
**Description:** Returns structured JSON for the side-by-side UI comparison matrix.
**Response:**
```json
[
  {
    "name": "ToolA",
    "category": "String",
    "pros": ["String"],
    "cons": ["String"],
    "pricing_tier": "String",
    "logo_url": "String"
  }
]
```

## 2. WebSocket Endpoints

### Stream Market Data
**Endpoint:** `WS /ws/market-data/{ticker}`
**Description:** Streams live price ticks to the frontend for a specific ticker (e.g., NVDA, Nifty).
**Message Payload:**
```json
{
  "ticker": "NVDA",
  "price": 125.40,
  "timestamp": "2023-10-27T10:00:00Z",
  "change_direction": "up"
}
```
